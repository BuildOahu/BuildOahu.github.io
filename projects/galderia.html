Scope of Work

Build a new bathroom and renovate one of the bedrooms.

Bathroom:

  • Demo shower tile, tub and basin
  • Install drain
  • Install shower valve
  • Basin and waterproofing
  • Grab bar backing
  • Tile bathroom and install shower glass
  • Trim door, window and base
  • Shower Glass
  • Install vanity & toilet and shower head
  • Mount Grab bars & accessory set
  • Finish paint

Bedroom:

  • Mount new closet doors and mirror
  • Install floor
  • Hang ceiling fan
  • Trim window, door and closet mirror
  • Change light switches and plates to white
  • Finish paint
  • Hang TV
